1.블록 레벨 텍스트 태그


: 혼자서 한 줄을 차지하여 너비가 100% 꽉 차게 표시되는 태그

'h태그': heading태그 - 제목을 표시하는 태그로 h1~h6 이 있다


가장 큰 제목태그

가장 작은 제목태그


2) 'p태그' : paragraph - 단락을 표시하는 태그로 줄바꿈없이 덱스트가 한 줄로 표시됨
"덱스트의 길이가 브라우저 창의 너미보다 길면 자동으로 줄이 바뀐다."

"br태그: 줄바꿈 처리는 태그로 닫는 태그가 없다"
'hr태그'(horizontal) :주제가 바뀌거나 할때 분기위전환을 위한 가로선 표시
닫는 태그가 없다.
'div태그'(division): 영역을 나누는 블록 태그

4. 표를 만드는 태그

table 태그, tr(table row) 태그, td(table data) 태그
행과 열의 형태로 이루어진 표의 형태를 만들 때 사용하는 태그
tr: 행에 해당하는 태그
td: 행 내부의 열에 해당하는 셀 태그
th(table head): 행 내부의 제목에 해당하는 셀 태그

테이블태그를 제목(thead, 본문(tbody), 요약(tfoot)의 구조로 표현 할 수 있다.

caption 태그: 표에 사용하는 표제목 표시 태그
표상단 가운데에 위치한다.

꺽쇠로 표현하는 태그: 여는 태그와 닫는 태그가 있고, 여는 태그만 사용하는 태그도 있다.
html: 웹문서를 만드는 언어 웹문서의 기본골격(명사)
Css(Cascading Style Sheet): 웹문서를 보기좋게꾸미는 디자인표현(형용사/부사)
javascript/jQuery: 프로그래밍적 요소로 사용자의 행위에 의한 동작(동사)
